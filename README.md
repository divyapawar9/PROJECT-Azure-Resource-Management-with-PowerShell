🚀 PROJECT : Azure Resource Management with PowerShell


📌 Project Title

Azure Resource Management Basics

📖 Overview

This project demonstrates basic Azure resource management using PowerShell commands. It includes retrieving virtual machine details and understanding how scripting can help in cloud administration tasks.

🛠️ Tools Used
Microsoft Azure
Azure PowerShell
Azure Portal

⚙️ Step-by-Step Implementation

1. Install Azure PowerShell
Open PowerShell
Run:
Install-Module -Name Az -AllowClobber -Scope CurrentUser

2. Connect to Azure
Connect-AzAccount

3. Get Resource Groups
Get-AzResourceGroup

4. Get Virtual Machines
Get-AzVM

5. Get VM Status
Get-AzVM -Status

6. Get Specific Resource Details
Get-AzResource

📊 Outcome:

Retrieved Azure resource and VM details using PowerShell
Understood basic scripting for cloud administration
Gained foundational knowledge of automation concepts
